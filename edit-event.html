    About platform

    Online Event Platform is a dApp deployed on the Aurora Testnet EVM chain where the platform organizer can host multiple online events with some options, one of which is an option that allows to watch the live stream of event only those users who have a special pass - "NFT Events Pass" . At the moment, this pass can be minted on dapp for free.
